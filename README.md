# CountdownTimer.dk - Gratis Dansk Countdown Timer ⏱️

Tæl ned til din store dag - bryllup, fødselsdag, ferie eller nytår.

## Features

- ✅ Live countdown til enhver dato/tid
- ✅ 7 smukke temaer (bryllup, fødselsdag, baby, ferie, etc.)
- ✅ Del countdown via link (URL params)
- ✅ Hurtige countdowns (1 time, 24 timer, nytår)
- ✅ Mobil-venligt responsive design
- ✅ Ingen login, ingen cookies, ingen data gemt

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS

## Sharing

Countdowns kan deles via URL parametre:
- `t` = titel
- `d` = dato (ISO format)
- `th` = tema (0-6)

Eksempel: `?t=Vores%20bryllup&d=2025-06-15T14:00&th=0`
